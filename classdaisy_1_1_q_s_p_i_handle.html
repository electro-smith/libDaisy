daisy::QSPIHandle Class Reference
LIBDAISY » PERIPHERAL » SERIAL

Detailed Description

Driver for QSPI peripheral to interface with external flash memory.
Currently supported QSPI Devices:
IS25LP080D

#include <qspi.h>
